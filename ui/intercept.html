LTE/4G usa NAS entre UE e MME. IMSI pode ser exposto no Attach Request inicial.
NAS: ATTACH REQUEST
Se o UE não tiver GUTI válido, envia IMSI em texto plano. IMSI Catcher 4G explora este comportamento.
IDENTITY REQUEST
MME falso emite Identity Request após Attach com GUTI inválido forçando IMSI em plaintext.
5G NR — SUCI
